38 /*! \brief This implements a bar graph with bars starting at \f$ y=0 \f$ to \f$ y=f(x) \f$
39  \ingroup jkqtplotter_barssticks
40 
41  This class plots a bargraph. This image explains the parameters:
42 
43  \image html bargraph_basics.png
44 
45  By default the sift parameter is, so the bar is centered at the x-value. The width is 0.9,
46  so adjacent bars are plotted with a small distance between them. It is possible to use these two parameters
47  to plot multiple bars for every x-value, by having on JKQTPStepHorizontalGraph object per
48  set of bars that belong together. For example for three bars per x-value one would set:
49  \verbatim
50  width=0.3
51  shift=-0.3 / 0 / +0.3
52  \endverbatim
53  This results in a bargraph, as shown here:
54 
55  \image html plot_bargraphverplot.png
56 
57 
58  You can use JKQTPlotter::addHorizontalBargraph() to add a series of bargraphs, where the width and shift are determined
59  automatically. The y-columns are given as a QVector<int> to this function.
60 
61 
62  */
