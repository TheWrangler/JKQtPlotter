#define LIB_EXPORT
 

Detailed Description

This file allows to export functions and classes from this directory in a dynamic/shared lib and to append the correct __declspec() declaration on windows. If you want to export one, make sure you have set

#include "jkqtplottertools/jkqtp_imexport.h"
class LIB_EXPORT exportedClassName {
...
};
LIB_EXPORT void exportedFunctionName();

Then when compiling define the preprocessor symbol LIB_IN_DLL when exporting AND using the classes/functions. When compiling the classes for export, also define the preprocessor symbol LIB_EXPORT_LIBRARY .
